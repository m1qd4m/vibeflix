## 🚀 Quick Start

### 1. Prerequisites

```bash
flutter --version   # Requires Flutter 3.x stable
dart --version      # Requires Dart 3.x
```

### 2. Clone & Install

```bash
git clone https://github.com/yourusername/vibeflix.git
cd vibeflix
flutter pub get
```

### 3. Configure APIs

Edit the `.env` file in the project root:

```env
TMDB_API_KEY=your_tmdb_key_here
GEMINI_API_KEY=your_gemini_key_here
YOUTUBE_API_KEY=your_youtube_key_here
```

**Getting API Keys (all free):**
- **TMDB**: https://www.themoviedb.org/settings/api — free, instant
- **Gemini**: https://aistudio.google.com/app/apikey — free (15 req/min, 1M tokens/day)
- **YouTube**: https://console.cloud.google.com → YouTube Data API v3 — free tier

> ✅ The app works **without any AI key** — it uses smart rule-based fallback logic automatically.
> ✅ Gemini 2.0 Flash is **completely free** with no credit card needed.

### 4. Firebase Setup

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure Firebase (creates firebase_options.dart and google-services.json)
flutterfire configure
```

This creates:
- `lib/firebase_options.dart` ✅
- `android/app/google-services.json` ✅

**Firebase Console setup:**
1. Create project at https://console.firebase.google.com
2. Enable Authentication → Email/Password + Google
3. Enable Firestore Database → Start in production mode
4. Add Android app with package name `com.vibeflix.app`

### 5. Run the App

```bash
# Debug
flutter run

# Release APK
flutter build apk --release

# App Bundle (Play Store)
flutter build appbundle
```

## 🔐 Firestore Security Rules

Paste in Firebase Console → Firestore → Rules:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;

      match /watchHistory/{movieId} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
      match /watchlist/{movieId} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
    }
  }
}
```

## 🧠 AI Chat Logic (Google Gemini — FREE)

```
User: "Something like Inception"
        ↓
Gemini 2.0 Flash extracts:
{
  "genres": ["Science Fiction", "Thriller"],
  "keywords": ["mind-bending", "dream", "heist"]
}
        ↓
TMDB /discover/movie with genre IDs + keyword IDs
        ↓
Returns matching movies
```

**Free tier**: 15 requests/min, 1,000,000 tokens/day — more than enough.  
**No API key?** Smart rule-based fallback handles 30+ patterns ("like Inception", "feel good", "scary", etc.) with zero API calls.

## 🔧 Troubleshooting

**Gradle build fails:**
```bash
cd android && ./gradlew clean
flutter clean && flutter pub get
```

**Firebase not initializing:**
- Ensure `google-services.json` is in `android/app/`
- Run `flutterfire configure` again

**Images not loading:**
- Check TMDB API key in `.env`
- Verify internet permission in `AndroidManifest.xml`
